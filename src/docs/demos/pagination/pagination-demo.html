Usability

The usability examples below illustrate the power of the visible-page-buffer attribute. As the user clicks through the pages, the width of the pagination component never shifts, allowing them to page through the results without ever moving their mouse. Not only can this be done 1 by 1 with the next button, but results can be stepped through multiple pages at a time using the buttons after/before the ellipses without any mouse movement (not available with the third demo, where visible-page-buffer is set to 0).
